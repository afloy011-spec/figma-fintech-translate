# Fintech Translator — как попробовать плагин

Короткая инструкция для коллег: что это, как установить и с чего начать.

---

## Что делает плагин

- Сканирует текст в выбранных **фреймах** в Figma.
- Переводит **с английского** на **ES / IT / FR / DE / PT** (можно выбрать языки).
- Даёт **просмотр и правку** перед вставкой в макет.
- Создаёт копии фреймов с переводом (**`Имя [ES]`**, **`[IT]`** и т.д.) или обновляет уже существующие языковые фреймы (**Sync**).
- Есть режим **Smart Fit**: перенос по ширине и при необходимости уменьшение шрифта, чтобы текст не вылезал из вёрстки.

**Движок перевода:** бесплатный (Google Translate + запасной канал) или **Pro** (своим ключом OpenAI — качество выше).

---

## Что понадобится

| Требование | Зачем |
|------------|--------|
| **Figma Desktop** | Импорт плагинов из режима **Development** в веб-версии ограничен |
| **Node.js** (LTS, например 18+) | Одна команда сборки: `npm install` и `npm run build` |
| Папка с плагином | Архив от автора или `git clone` репозитория |

Ключ OpenAI **не обязателен** — можно начать с бесплатного режима.

**Как передать папку коллеге:** заархивируйте весь проект плагина. Папку `node_modules` можно **не класть в архив** — у коллеги она появится после `npm install` (архив будет легче).

## Установка (один раз)

1. Распакуйте папку плагина (или клонируйте репозиторий) в удобное место, например:
   `…\figma-fintech-translate`

2. В терминале перейдите в эту папку и выполните:

   ```bash
   npm install
   npm run build
   ```

   Должны появиться/обновиться файлы в папке **`dist/`** (`code.js`, `ui.html`).

3. Откройте **Figma Desktop** → меню **Plugins** → **Development** → **Import plugin from manifest…**

4. Выберите файл **`manifest.json`** в корне папки плагина (рядом с `package.json`).

5. Запуск: **Plugins** → **Development** → **Fintech Translator** (или как указано в манифесте).

> Если меняли код после установки — снова выполните `npm run build` и при необходимости **переимпортируйте** плагин из того же `manifest.json`.

## Быстрый старт в файле

1. Выделите **один или несколько фреймов** с текстом на странице.
2. Откройте плагин.
3. Выберите языки и режим (**Free** или **Pro** + ключ, если нужен OpenAI).
4. Нажмите **Scan Selection** — проверьте список найденных строк.
5. **Translate** → при необходимости отредактируйте в **Review**.
6. **Apply to Frames** — появятся копии с переводом (или обновятся целевые фреймы, если вы используете **Sync** и уже есть фреймы с именами вида `База [ES]`).

Опционально в блоке **Advanced**: **Smart Fit** (перенос, отступы, авто-масштаб шрифта) и **Sync from Source** (обновить все языки с одного исходного фрейма).

---

## Если что-то не работает

- После изменений в коде снова: `npm run build`.
- Ошибка при запуске: **Plugins** → **Development** → откройте **консоль разработчика** у плагина и посмотрите **первое красное сообщение**.
- Сеть: в `manifest.json` уже перечислены разрешённые домены для API перевода и OpenAI; в корпоративной сети иногда нужен VPN или политика IT.

---

## Конфиденциальность (коротко)

- В режиме **Pro** ключ OpenAI хранится в **Figma clientStorage** и уходит только в **OpenAI** (не в сторонние серверы автора плагина).
- Не публикуйте скриншоты с ключом и не коммитьте ключ в git.
